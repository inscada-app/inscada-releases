# inSCADA Releases

Official release feed for [inSCADA](https://inscada.com) — Industrial SCADA Platform.

## Download

Each release contains the inSCADA application JAR file. Download the latest release from the [Releases](https://github.com/inscada-app/inscada-releases/releases) page.

## Installation

1. Download the latest `app-*.jar` from the releases page
2. Replace the existing JAR file on your server
3. Restart the inSCADA service
